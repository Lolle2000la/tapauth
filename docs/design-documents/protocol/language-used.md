# Language used

## Terminology

* Client: The software or device that is requesting authentication from the server.
* Server: The software or device that is providing authentication services to the client. A single network may have multiple servers and a single server may serve multiple clients.
* User: The person who is using the client and/or server software or device.
* Authentication request: A message sent by the client to the server, requesting authentication.
* Authentication grant: A message sent by the server to the client, granting authentication. This will be used by the client to unlock the user account (= log in).
